## 🚀 Como rodar localmente

1. Clone este repositório:
```bash
git clone https://github.com/seu-usuario/work-shop.git
```

2. Navegue até a pasta do projeto:
```bash
cd work-shop
```

3. Instale as dependências:
```bash
npm install
```

4. Execute o projeto:
```bash
ng serve
```

5. Acesse no navegador: `http://localhost:4200`
